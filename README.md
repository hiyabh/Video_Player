## ✨ תכונות

### 🎨 **עיצוב מודרני 2025**
- 🌈 **Glassmorphism UI** - ממשק משתמש עם אפקטי זכוכית מטושטשים
- 🌓 **Dark/Light Mode** - מעבר חלק בין מצבי תצוגה
- 🎭 **אנימציות חלקות** - מיקרו-אינטראקציות עם CSS transitions
- 📱 **Responsive Design** - מותאם לכל הגדלי מסך
- 🎯 **RTL Support** - תמיכה מלאה בעברית

### 🎥 **בקרת וידאו מתקדמת**
- ▶️ **Play/Pause** - הפעלה ועצירה עם לחיצה על הוידאו
- ⏪ **Rewind** - הרצה אחורה במהירות רגילה (1x)
- ⏩ **Fast Forward** - הרצה קדימה מהירה (5x-10x)
- 🔄 **Frame Navigation** - ניווט פריים-פריים (30fps)
- 📊 **Progress Bar** - סרגל התקדמות אינטראקטיבי עם drag support

### 🔧 **כלי עריכה וטרנספורמציה**
- 🔍 **Zoom In/Out** - זום חכם עם גלגלת העכבר או כפתורים
- 🔄 **Rotation** - סיבוב מדויק של 5° בכל כיוון
- 🎯 **Position Control** - הזזה חופשית בכל הכיוונים
- 🖱️ **Drag & Drop** - גרירה עם עכבר לשינוי מיקום
- 🔁 **Reset** - איפוס מיידי לתצוגה המקורית
- 📍 **Real-time Info** - תצוגת זום, סיבוב ומיקום בזמן אמת

### 📤 **העלאת קבצים**
- 📁 **Drag & Drop Zone** - גרירת קבצים ישירות למסך
- 📹 **Format Support** - תמיכה ב-MP4, WebM, OGG
- 📊 **File Info Display** - הצגת שם ומשקל הקובץ
- ⚡ **Quick Upload** - טעינה מהירה וחלקה

### ⌨️ **קיצורי מקלדת**
- 💡 **Help Modal** - מודל עזרה מעוצב עם כל קיצורי הדרך
- 🎹 **Full Keyboard Support** - שליטה מלאה מהמקלדת
- ❓ **Press ?** - פתיחה מהירה של מודל העזרה

## ⌨️ קיצורי מקלדת

### 🎮 בקרת הפעלה

| מקש | פעולה |
|-----|-------|
| `Space` / `K` | נגן / עצור |
| `→` | קדימה 5 שניות |
| `←` | אחורה 5 שניות |
| `J` | אחורה 10 שניות |
| `L` | קדימה 10 שניות |
| `0` | חזרה להתחלה |

### 🖼️ תצוגה

| מקש | פעולה |
|-----|-------|
| `F` | מסך מלא |
| `M` | השתק / בטל השתקה |
| `?` | פתח עזרה |
| `Esc` | סגור מודל |

### 💡 טיפים
- 🖱️ **גלגל עכבר** על הוידאו = זום חכם
- 🖱️ **גרור** את הוידאו להזזה
- 🖱️ **החזק כפתור** למשוך ברציפות (זום, הזזה)
- 🖱️ **לחץ על וידאו** להפעלה/עצירה

## 📦 התחלת עבודה

### 🌐 שימוש מקוון
פשוט פתח את [הדגמה החיה](https://hiyabh.github.io/Video_Player/index.html)!

### 💻 התקנה מקומית

```bash
# שלב 1: שכפול המאגר
git clone https://github.com/hiyabh/Video_Player.git

# שלב 2: כניסה לתיקייה
cd Video_Player

# שלב 3: פתיחת index.html בדפדפן
# אופציה 1: פתיחה ישירה
open index.html

# אופציה 2: עם שרת מקומי (מומלץ)
python -m http.server 8000
# גלוש ל: http://localhost:8000
```

### 🎯 שימוש מהיר

1. **העלה וידאו** 📤
   - גרור קובץ לאזור ההעלאה
   - או לחץ על "העלאת וידאו" ובחר קובץ

2. **נגן וערוך** 🎬
   - השתמש בכפתורי הבקרה
   - גרור את הוידאו להזזה
   - השתמש בגלגלת לזום

3. **קיצורי מקלדת** ⌨️
   - לחץ `?` לעזרה
   - השתמש ב-`Space` לנגן/עצור
   - `F` למסך מלא
